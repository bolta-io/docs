## 2. 발행 상태

- 승인 대기 : 고객이 발행 정보 입력을 완료한 상태
- 발행 대기 : 요청한 발행날짜가 도래하지 않아 발행 예정인 상태
- 발행 완료 : 발행 승인이 완료되어 국세청에 정보가 전송완료된 상태
- 발행 거절 : 발행 현황에서 거절을 눌러 발행을 거절한 상태
- 발행 실패 : 발행이 어떠한 이유로 실패한 상태
